# Modification Notice

Maestro is derived from OpenAI Symphony.

The original OpenAI Symphony work is attributed in `NOTICE` and remains subject
to the inherited Apache-2.0 attribution and notice requirements preserved in
`LICENSES/Apache-2.0.txt`.

Maestro modifications include, but are not limited to:

- Rebranding public project references from Symphony-derived naming to Maestro.
- Extending the original Linear-focused orchestration model toward multiple trackers, repository providers, agent providers, workflow profiles, runtimes, tool bridges, and evidence surfaces.
- Adding TAPD and CNB provider support alongside the original GitHub/Linear-oriented flows.
- Adding broader agent-provider support for Codex, Claude Code, and OpenCode.
- Adding runtime, worker-daemon, dynamic-tool, observability, and dashboard changes for operating agentic engineering workflows at team scale.
- Relicensing Maestro modifications under the GNU Affero General Public License version 3 (AGPL-3.0-only), while preserving attribution for inherited Apache-2.0 material.

This file is intended as a repository-level prominent notice that the derivative
work has been modified from the original OpenAI Symphony project.
